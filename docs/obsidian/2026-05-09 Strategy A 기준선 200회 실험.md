## 이날 해결된 이슈

### IP Rotation 커널 패닉 → 자동 재부팅
- **원인**: `svc data disable` 호출 시 활성 naver.com TCP 연결이 살아있으면 모뎀 드라이버 패닉
- **해결**: `clearPage()` — about:blank 로드 후 3초 대기 → 모든 TCP 연결 정리 후 rotation
- **추가**: `am force-stop com.sec.android.app.sbrowser` — 삼성 인터넷 백그라운드 연결 차단

### WebView에서 삼성 인터넷 열림
- **원인**: `shouldOverrideUrlLoading` 미구현으로 intent:// 등 비http 스킴이 외부 앱 호출
- **해결**: intent:// 및 비http 스킴 차단 구현
